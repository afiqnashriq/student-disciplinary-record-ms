# Secure Student Disciplinary Record Management System

## Description
A PHP-based system to manage student disciplinary records. Supports multiple user roles: admin, staff, and student.

## How to run
1. Clone this repo
2. Install dependencies via Composer
3. Configure email credentials locally in `config_local.php` (not included)
4. Import database SQL (see `database.sql`)
5. Run on local server (XAMPP / WAMP)

## Notes
- Email credentials are placeholders for GitHub safety.
